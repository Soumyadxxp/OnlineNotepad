# 📖 Overview

Online Notepad is a browser-based text editor that provides a clean and distraction-free writing experience.

Unlike traditional text editors, this application runs entirely in the browser and automatically saves your work using Local Storage, ensuring that your notes remain available even after refreshing the page.

The project demonstrates:

* DOM Manipulation
* Event Handling
* Local Storage
* Dynamic UI Rendering
* Responsive Web Design
* JavaScript Application Development
